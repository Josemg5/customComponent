# Listado de Herramientas y Librerías

Resumen de las tecnologías empleadas para el diseño, desarrollo y prueba del componente SwitchButton.

## 1. Herramientas de Diseño
* **JavaFX (Diseño programático):** Creación de la interfaz mediante código Java puro (uso de `Rectangle`, `Circle` y efectos), sin depender de editores visuales externos como SceneBuilder.

## 2. Herramientas de Desarrollo
* **IntelliJ IDEA:** IDE utilizado para la escritura de código, refactorización y ejecución de tareas.
* **Gradle:** Herramienta de automatización para gestionar dependencias, compilar y generar el archivo `.jar`.
* **Git y GitHub:** Control de versiones para el historial de cambios, gestión de Releases y alojamiento de la documentación.

## 3. Herramientas de Prueba
* **JUnit 5:** Framework para verificar la lógica interna del componente (getters, setters y propiedades).
* **TestFX:** Herramienta para simular la interacción del usuario (clics) y validar la respuesta visual de la interfaz.

## 4. Librerías
* **JavaFX Graphics:** Proporciona los nodos gráficos y las transiciones necesarias para las animaciones.
* **TestFX-JUnit5:** Permite la integración de las pruebas de interfaz gráfica dentro del motor de JUnit 5.

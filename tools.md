# Herramientas Utilizadas

Aquí detallo los programas y librerías que he usado para desarrollar esta práctica.

## Desarrollo
* **IntelliJ IDEA:** El entorno de desarrollo (IDE) donde he escrito todo el código.
* **JavaFX:** La librería gráfica de Java. Es la que permite crear botones, ventanas y hacer que el círculo se mueva (animación).
* **Gradle:** Es el gestor del proyecto. Se encarga de descargar las librerías necesarias automáticamente y de crear el archivo final (`.jar`).

## Pruebas (Testing)
* **JUnit 5:** Framework para hacer pruebas unitarias. Lo he usado para comprobar que la lógica interna funciona (por ejemplo, que el booleano cambia de true a false).
* **TestFX:** Librería específica para probar interfaces gráficas. Simula a un usuario haciendo clic con el ratón sobre el componente para ver si reacciona.

## Documentación y Control
* **Git y GitHub:** Para guardar el historial de cambios del código y subirlo a la nube.
* **GitHub Pages:** La herramienta que convierte estos archivos de texto en la página web que estás viendo ahora mismo.

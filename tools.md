# Herramientas

## Desarrollo
* **IntelliJ IDEA:** IDE usado para escribir el código.
* **JavaFX:** Librería para crear la interfaz y la animación.
* **Gradle:** Herramienta para gestionar librerías y crear el .jar.

## Pruebas
* **JUnit 5:** Usado para probar la lógica del botón.
* **TestFX:** Usado para simular clics y probar la interfaz.

## Otros
* **Git/GitHub:** Para guardar versiones y subir el código.
* **GitHub Pages:** Para generar esta web de documentación.
